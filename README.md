# ABS PRIYA OIL MILL - Premium Marachekku Oil Website

🌿 **100% Pure Traditional Wood Pressed Oils**

A modern, premium, fully responsive business website for ABS PRIYA OIL MILL - Traditional Marachekku (Wood Pressed) Oil manufacturer in Tamil Nadu, India.

## 🎨 Features

### Core Features
- ✅ Premium modern design with green & gold theme
- ✅ Fully responsive (Mobile, Tablet, Desktop)
- ✅ Bilingual support (English & Tamil)
- ✅ Dark/Light mode toggle
- ✅ Smooth animations & transitions
- ✅ SEO optimized
- ✅ Performance optimized

### Pages
1. **Home** - Hero banner, intro, CTA
2. **About Us** - Company story & mission
3. **Products** - Product cards with pricing
4. **Why Choose Us** - Key benefits
5. **Benefits** - Health benefits of oils
6. **Gallery** - Images & process photos
7. **Reviews** - Customer testimonials
8. **Contact** - Address, map, contact form
9. **Online Order** - WhatsApp & Payment gateway
10. **FAQ** - Common questions

### Integration Features
- 🔗 WhatsApp Order Button (Floating & Fixed)
- 📞 Click to Call functionality
- 💳 Razorpay Payment Gateway
- 🗺️ Google Maps integration
- 📧 Contact form with email notifications
- 📱 Responsive design

## 🛠️ Technology Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **State Management**: Zustand
- **Icons**: Lucide React
- **Sliders**: React Slick
- **Notifications**: React Hot Toast
- **HTTP Client**: Axios

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/dharshika6-crypto/abs-priya-oil-mill-.git

# Navigate to project
cd abs-priya-oil-mill-

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env.local

# Update .env.local with your values
```

### Development

```bash
# Start development server
npm run dev

# Open http://localhost:3000
```

### Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```
